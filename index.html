FRONTEND Y BACKEND

Hecho por: Alicia Navarro Pino

fronted

El front end es aquello que ven los usuarios e incluye elementos visuales, como botones, casillas de verificación, gráficos y mensajes de texto. Permite a los usuarios interactuar con la aplicación. El back-end son los datos y la infraestructura que permiten que la aplicación funcione.

una aplicación front-end, comúnmente conocida como la interfaz de una aplicación, es la capa o elemento que el usuario tiene la capacidad de usar, ver e interactuar a través de botones, imágenes, elementos interactivos, menús de navegación y texto.

BACKEND

backend es la parte del desarrollo web que se encarga de que toda la lógica de una página web funcione. Se trata del conjunto de acciones que pasan en una web pero que no vemos como, por ejemplo, la comunicación con el servidor.

A veces denominado servidor, el back end de la aplicación administra la funcionalidad general de la aplicación web. Cuando el usuario interactúa con el front end, la interacción envía una solicitud al back end en formato HTTP. El backend procesa la solicitud y devuelve una respuesta.
